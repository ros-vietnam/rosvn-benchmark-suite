## 1. Overview

**ROS Vietnam Robotics Benchmark Suite** is the official benchmarking repository of the **ROS Vietnam** community.

This repository provides a **neutral, transparent, and reproducible framework** to evaluate robotics algorithms across:
- Simulation
- Metrics
- Experimental protocols

It is designed to ensure **fair comparison**, **scientific integrity**, and **long-term reproducibility**.

> This repository does not promote algorithms.  
> It evaluates them.

---

## 2. Why Benchmarking Matters

Without standardized benchmarks:
- Results are not comparable
- Claims are not verifiable
- Progress is unclear
- Research becomes anecdotal

This repository exists to:
- Separate performance from marketing
- Encourage honest evaluation
- Enable reproducible robotics research
- Establish community-wide standards

> *Benchmarks are the backbone of scientific progress.*
